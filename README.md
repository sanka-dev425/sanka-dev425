## Active Projects

**ReconLK** — Multi-Tenant SaaS Financial Control Platform
> Enterprise-grade financial operating system for Sri Lankan SMEs. 
> Inventory management, invoicing, payment tracking, and bank 
> reconciliation — unified, auditable, and explainable.
> Stack: React · Node.js · PostgreSQL 

**AI Insurance Recommendation System** — Prototype
> Multilingual AI platform recommending life and health insurance 
> in Sinhala, Tamil, and English using Explainable AI and NLP. 
> Built to address financial accessibility in Sri Lanka.
> Stack: Python · NLP · Explainable AI (XAI) · Machine Learning
